## 개요

설치 스크립트의 기본 설치 경로를 시스템 경로가 아닌 유저 로컬 경로로 변경합니다.

## 현재 상태

| OS          | 현재 기본 경로               |
| ----------- | ---------------------------- |
| macOS/Linux | `/usr/local/bin` (sudo 필요) |
| Windows     | `$LOCALAPPDATA\Programs\jd`  |

## 변경 후

| OS          | 변경 후 기본 경로         |
| ----------- | ------------------------- |
| macOS/Linux | `$HOME/.local/bin`        |
| Windows     | `$USERPROFILE\.local\bin` |

## 수정 대상 파일

1. `install.sh` - 기본 경로 변경
2. `install.ps1` - 기본 경로 변경
3. `README.md` - 문서 업데이트
